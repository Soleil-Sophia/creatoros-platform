## 🚫 What NOT to Do Next

Based on user feedback and V40 learnings:

1. ❌ **No new große Sektionen erfinden**
   - The page has enough structure
   - More sections = longer, not better

2. ❌ **No Grundpositionierung wieder anfassen**
   - System + Reuse + Assets + modular creator infrastructure is set
   - Don't touch the foundation again

3. ❌ **No zu viele neue Module auf Homepage**
   - Homepage = Wedge + Credibility
   - Not entire future universe

4. ❌ **No Copy aus Enterprise-Gewohnheit aufblasen**
   - Less: "innovations-driven platform language"
   - More: Concrete product mechanics

5. ❌ **No jeden Zweifel mit Umstrukturierung beantworten**
   - You're in Polish-Modus, not Identitätskrisen-Modus

---

## ✅ What to KEEP Doing

   - "system saves nothing"

## 🧪 Next Phase: Testing

### Phase B — Review (Recommended)

#### Test 1: 5-Second Understanding
Ask 2-3 people after 5-10 seconds:
- What does the page sell?
- Who is it for?
- How is it different from normal AI chat?

**Target Answer:**
> "A modular system for creators that turns ideas into reusable content assets instead of just chat answers."

**Red Flag:**
> "AI Content Tool"

#### Test 2: CTA Clarity
Ask:
- What would you click?
- What do you expect behind "Start Creating"?
- What do you expect behind "Explore Modules"?
- Is "Launch Content OS" clear?

**Goal:** CTA should create no thinking pause

#### Test 3: Scroll Fatigue on Mobile
Ask:
- Did it get monotonous anywhere?
- Was there a spot where you got impatient?
- Which section was strongest?
- Which was most unnecessary?

**Suspect Spots:**
- Outputs Card Stack (now fixed with grouping)
- Footer / lower navigation
- Ecosystem Preview

#### Test 4: Credibility Test
Ask:
- Does this sound like a real product or a concept?
- Which section convinced you?
- Which section still sounded like marketing?

**Goal:**
> "Sounds like a real product"

**Not:**
> "Sounds like a cool vision"

#### Test 5: Desktop Check
- Does Hero work as strongly on desktop?
- Are spacings too empty or too tight?
- Does Workflow Preview carry on big screen?
- Does Comparison Table get better or harder on desktop?
