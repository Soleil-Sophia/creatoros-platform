# CreatorOS Stabilization Sprint 1

**Status:** Active  
**Scope:** `BrandOS → ContentOS → LaunchOS`

## Objective

Prove one honest lifecycle:

```text
Brand Context → Content Asset → Launch Plan
```

## Required Scenario

1. Create or load an extended Brand Profile.
2. Save and reload it.
3. Open ContentOS and verify the same brand context.
4. Generate, review, and save an asset.
5. Continue to LaunchOS.
6. Confirm the saved asset appears.
7. Create and save a Launch Plan.
8. Reload and recover the plan.
9. Record commit, deployment, evidence, and limitations.
