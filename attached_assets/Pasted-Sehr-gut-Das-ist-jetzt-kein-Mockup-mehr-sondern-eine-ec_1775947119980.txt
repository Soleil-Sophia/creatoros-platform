Nächster Schritt: Prompt 6 mit Gumroad-Flow

Refine the existing creatoros-site marketing app so the “Get the Kit” CTA leads into a real purchase-ready MVP flow using Gumroad as the external checkout destination.

Goal:
Make Creator Clarity Kit feel actually purchasable now, with a clean external checkout path and clear delivery expectations.

Constraints:
	•	Do not build a custom checkout inside the site
	•	Use Gumroad as the external checkout destination
	•	Keep the site premium, minimal, and editorial
	•	Do not add fake urgency, fake discounts, or fake proof
	•	Maintain CTA discipline:
	•	Primary product CTA: “Get the Kit”
	•	Secondary brand CTA: “Explore CreatorOS”
	•	Lead capture CTA: “Join Early Access”

Implement the following:
	1.	Central checkout config
Create a single config constant for the Gumroad checkout URL, for example:
CREATOR_CLARITY_KIT_CHECKOUT_URL

Use this same destination everywhere the primary CTA appears.
	2.	Connect all primary CTAs
Update every “Get the Kit” button across:

	•	Home
	•	Product
	•	Offers
	•	CTA sections
to point to the same Gumroad checkout URL.

No dead links. No inconsistent routes.
	3.	External checkout behavior
Since Gumroad is external:

	•	open checkout intentionally and cleanly
	•	optionally open in a new tab
	•	if using an external-link cue, keep it subtle and premium
	•	do not overdecorate it

	4.	Product purchase block polish
On /product, make the purchase block feel final and real:

	•	Product name
	•	€24 price
	•	One-time purchase
	•	Instant access after checkout
	•	Includes:
	•	Notion Template
	•	Workbook
	•	Prompt Assist
	•	Start Here Guide
	•	Primary CTA: Get the Kit

Keep this clean, premium, and non-cluttered.
	5.	Post-purchase expectation block
Add or refine a short section on /product:

Headline:
What happens after purchase

Body:
After checkout, you’ll get instant access to the full Creator Clarity Kit, including the Notion Template, Workbook, Prompt Assist, and Start Here Guide.

Optional bullets:
	•	Instant access
	•	One-time purchase
	•	No subscription
	•	Built for fast clarity, not overwhelm

	6.	Offers page clarity
Make the offers page clearly distinguish between:

	•	Available now:
	•	Creator Clarity Kit
	•	€24
	•	Get the Kit
	•	Not yet live:
	•	future tools or modules
	•	Join Early Access

The page should clearly communicate what can be bought today versus what is only upcoming.
	7.	Microcopy near CTAs
Where appropriate, add subtle reassurance copy near “Get the Kit”:

	•	€24 — one-time purchase
	•	Instant access after checkout

Keep this tasteful and minimal.
	8.	Gumroad-ready flexibility
Implement the checkout URL in a way that it can easily be replaced later with the real Gumroad product link without changing button components individually.

Goal:
Make creatoros-site feel like a real premium digital product website with a believable live purchase path using Gumroad.—
